## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/igor-marchi/proffy podcastr
# Install dependencies
$ cd yarn or npm -i
# Start server
$ cd yarn server
# Start project
$ yarn or npm dev
```
